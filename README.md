# Dragon Sales - AI研修営業トークスクリプト

AI研修会社の営業トークスクリプトをインタラクティブなフローチャートで可視化するWebアプリケーション。

## 機能

- **インタラクティブなフローチャート**: ReactFlowを使用した営業フロー可視化
- **詳細パネル**: ノードクリックでトークスクリプト、チェックポイント、Q&Aを表示
- **練習モード**: 営業役/顧客役を切り替えながらロールプレイ練習
- **進捗バー**: 営業フェーズ（開始→ヒアリング→提案→クロージング→フォローアップ）の可視化
- **キーボードショートカット**: Escapeで閉じる、矢印キーでフェーズ移動

## 技術スタック

- **Framework**: Next.js 14 (App Router) + TypeScript
- **UI**: Tailwind CSS + shadcn/ui
- **フローチャート**: ReactFlow (@xyflow/react)
- **アニメーション**: Framer Motion

## デザイン

ダーク&モダンテーマ（黒基調、ネオン風アクセント）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 本番サーバー起動
npm start
```

## 営業フェーズ

1. **開始**: アポイント確認・アイスブレイク
2. **ヒアリング**: 現状の課題ヒアリング・深堀り
3. **提案**: ソリューション提案
4. **クロージング**: 前向き/検討中の対応
5. **フォローアップ**: 次回アクション設定
